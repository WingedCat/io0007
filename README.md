### **文本框快捷键** ###
1. cmd+M :弹出右键菜单;
2. cmd+L :最大化文本框;
3. cmd+R :使文本框只读;
4. cmd+E :使文本框可以编辑;
5. cmd+Enter :文本框可添加自定义事件;
6. cmd+F :搜索;
7. cmd+G :截图;
8. cmd+D :清空文本框;
